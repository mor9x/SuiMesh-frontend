Own the Context. Verify the Action.

SuiMesh is the communication protocol layer for AI Agents on Sui, built for auditable coordination, traceable execution, and onchain memory.

Explore Protocol ↗ Developers
Why SuiMesh

Built for agent systems that need auditability, control, and durable memory.

SuiMesh keeps context intact, verifies actions before execution, and preserves memory as protocol infrastructure instead of leaving it to application logic.

01

Auditable coordination

Every handoff can leave a durable receipt, so teams can inspect how agents coordinated and why a decision path was taken.

02

Traceable coordination

Context no longer disappears between agents. SuiMesh preserves the handoff layer so multi-agent systems stay understandable.

03

Memory on Sui

Shared memory becomes durable protocol state, making replay, recall, and long-running agent continuity far more reliable.

Protocol Flow

From agent intent to verifiable record.

The protocol does not just move messages. It carries context, inserts verification, and records the resulting memory layer.

Execution Trace
Step 01
Capture context Agent state, input references, and operating constraints are packaged into a shared context envelope.
Step 02
Verify the action Intended execution is checked against policy, user intent, and permission rules before handoff.
Step 03
Commit shared memory Relevant outputs and state deltas become durable memory, suitable for inspection, replay, or onchain anchoring.
Step 04
Replay and inspect Operators, downstream agents, and governance surfaces can reconstruct what happened without relying on guesswork.
Layer One

Context integrity

Shared context should survive agent boundaries without losing provenance, approval state, or operational assumptions.

Layer Two

Verification before execution

Official AI systems need a control point between intent and action. SuiMesh introduces that layer natively.

Layer Three

Memory as infrastructure

Rather than treating memory as transient app state, SuiMesh allows collaborative memory to become an auditable protocol primitive.
